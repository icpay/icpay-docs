# Pay Button

The Pay Button component renders a compact, single-action checkout button for a fixed USD amount. Optionally show a ledger dropdown; if hidden, it uses `defaultSymbol`. {{ className: 'lead' }}

## Usage

<CodeGroup title="Pay Button">

```tsx {{ title: 'Next.js' }}
'use client'
import { useEffect, useRef } from 'react'
import '@ic-pay/widget'

export default function Page() {
  const ref = useRef<any>(null)
  useEffect(() => {
    if (!ref.current) return
    ref.current.config = {
      publishableKey: process.env.NEXT_PUBLIC_ICPAY_PK,
      amountUsd: 49.99,
      defaultSymbol: 'ICP',
      // Hide dropdown â†’ button shows selected symbol: "Pay $49.99 with ICP"
      showLedgerDropdown: false,
      // Modal progress bar is recommended for buttons
      progressBar: { enabled: true, mode: 'modal' },
    }
  }, [])
  return <icpay-pay-button ref={ref as any} />
}

## Configuration

- `amountUsd?: number`
- `cryptoOptions?: { symbol: string; label: string; canisterId?: string }[]`
- `showLedgerDropdown?: boolean` (default: false)
- `defaultSymbol?: string` (e.g., `'ICP'`)
- `buttonLabel?: string` (used if `amountUsd` not set)
- `progressBar?: { enabled?: boolean; mode?: 'modal' | 'horizontal' | 'vertical' | 'inline' }`

Notes:
- When `showLedgerDropdown` is false, the button uses `defaultSymbol` and displays it (e.g., "Pay $49.99 with ICP").
- When `showLedgerDropdown` is true, the button shows generic brand text (e.g., "Pay $49.99 with ICPay").
- For non-modal progress modes, the bar appears only during processing.
